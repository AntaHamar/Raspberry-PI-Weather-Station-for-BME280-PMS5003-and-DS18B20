# Raspberry-PI-Weather-Station-for-BME280-PMS5003-and-DS18B20
This code is for any project which uses these components: PMS5003, BME280 and DS18B20. Make sure to import the modules before running the code, or else it could crash. Further, actual finalised code coming soon! :)

An overview of what has happened so far...

File Name: FinalWeatherStationGUI.py - this file is a simulation for the BME280, PMS5003 and DS18B20. The numbers generated are not real, but are in fact random. The GUI comes with accessibility features such as VoiceOver, High-Contrast Mode and Zoom. Keybinds are listed on the GUI.

File Name: WilliamWeatherStation.py - this file is the actual code, but contains errors as it is unedited and has small issues. This code is for when the components are installed into your Raspberry PI micro-computer. Be careful with this file!

File Name: FinalWilliamWeatherStation - this is the actual, perfect file, with no errors (hopefully) and should run fine unless you have not installed something correctly such as the modules or have missed out a line of code

File Name: B280 - this file is just a testing file for the BME280 component

File Name: P5003 - this file is just a testing file for the PMS5003 component

File Name: DS18B2O - this file is just a testing file for the DS18B20 component
